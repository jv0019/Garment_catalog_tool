# Garment Catalogue Image Automation Tool

A Python desktop application that automates the preparation of product images for garment catalogues.  
The tool batch processes images by resizing them, removing backgrounds, adding sequential product codes, and overlaying brand logos.

This project was built to simplify repetitive image preparation tasks commonly required in garment wholesale and catalogue production.

---

## Problem

Garment wholesalers and catalogue creators often need to process hundreds of product images before publishing them. Typical manual tasks include:

- Resizing images to a standard catalogue format
- Removing image backgrounds
- Adding product codes or SKUs
- Branding images with company logos

Doing this manually in image editing software can take **hours of repetitive work**.

---

## Solution

This application automates the entire workflow through a simple desktop interface.

Users can select a folder of product images and automatically:

- Resize all images to a specified resolution
- Remove image backgrounds using AI segmentation
- Add sequential product codes
- Overlay a brand logo on each image
- Export the processed images to an output directory

The process reduces manual effort from **hours to minutes**.

---

---

## How It Works

1. Select the **Input Directory** containing images.
2. Select the **Output Directory** where processed images will be saved.
3. Select a **Logo Image** to overlay on the processed photos.
4. Configure:
   - Starting counter for product numbering
   - Text prefix and suffix
   - Image width and height
5. Click **Start Processing**.

The application will automatically process every image in the input folder.

---

## Example Output Text

Example product code generated by the tool:

```
A4P-25 SIZE-10X14 CODE-BGN
```

The numbering automatically increments for each processed image.

## Installation

Clone the repository:

```bash
git clone https://github.com/jv0019/Garment_catalog_tool.git
cd image_calalg_tool
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
python main.py
```
